# 免注册 C++/WinRT 示例

此存储库演示如何使用 [免注册 WinRT](https://blogs.windows.com/windowsdeveloper/2019/04/30/enhancing-non-packaged-desktop-apps-using-windows-runtime-components/) 来引用非打包桌面应用中的 C++/winrt 组件。

示例中演示了普通类、派生类和静态类的实现。
